2. Local Storage

Partition Table Format

MBR (master boot record)

自1980s，最多 4 個 primary partition，每個 partition 最大 2TB。

GPT (globally unique identifier)

自2000s，最多 128 個 partition，每個 partition 最大 18EB。 (GB→TB→PB→EB)

Disk Type

Basic disk

正常用法，可以分 partition。

Dynamic disk

可以在電腦正在運行的時侯設置 RAID，在 disk manager 入面 set。實際使用只比在 BIOS 設置的 RAID 慢少許。Dynamic disk 分的叫做 volume 而不是 partition。
Basic disk 可以轉成 Dynamic disk，但 Dynamic 就不能轉成 Basic。

File System

FAT

沒有任何 security，支持容量少。
(FAT 支持 2GB，FAT32 支持 2TB，exFAT 支持大於 2TB)

NTFS

● Reliability – 對於突然斷電的保護更好
● Permission
● Encryption
● DC
● Compression
● 可以 Extend or Shrink

ReFS

NTFS 的加强版，各項都强啲。但只可以 Extend。

Virtual Disk

.vhd

舊版

.vhdx

Windows 8 or later, Windows server 2012 or later, 你就可以用 .vhdx。支持更大容量，最大去到 64TB。

(可以改 sector size，即每一次最少讀多大 size 的內容，你可以選擇 512 bytes 到 64 KB。)

Volume Type

(用 Dynamic disk 才可以揀 volume type)

● Simple – 普通盤
● Spanned – 跨兩隻盤，最多可以跨到32隻
● Striped – RAID-0
● Mirrored – RAID-1
● RAID-5 – 浪費1/n，建基在 RAID-0 上，但有容錯 (有一隻做 partiy)
