Read Synology NAS HDD on Ubuntu

sudo -i
apt update
apt install -y mdadm lvm2

● 先安裝兩個 RAID 管理工具，mdadm 和 lvm2。
● "-y" 表示 all "Yes"。


mdadm -Asf && vgchange -ay

● 然後掛載所有從 Synology NAS 取出的硬碟。
