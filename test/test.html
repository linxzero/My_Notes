.reg

測試 H1 header

.reg 的撰寫

語法

RegistryEditorVersion

[RegistryPath1]
"DataItemName1"=DataType1:DataValue1
"DataItemName2"=DataType2:DataValue2

[RegistryPath2]
"DataItemName3"=DataType3:DataValue3

例子：

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon]
{{"DefaultUserName"="dc"}}
"DefaultPassword"="88888"
"AutoAdminLogon"="1"

字串(REG_SZ)類型不需要加 DataType，只要有引號引起來
其他類型則需要註明 DataType，但不用加引號

"DataItemName2"=hex(2):73,00,79,00,73,00,74,00
"DataItemName3"=hex(7):5c,00,44,00,65,00,76,00

版本

Windows 2000、Windows XP、Windows Server 2003 中是 "Windows Registry Editor Version 5.00"
Windows 98、Windows NT 4.0 則是 "REGEDIT4"

刪除

若要以 .reg 來刪除登錄機碼 (Key)，請在 RegistryPath 前面加上減號

[-HKEY_LOCAL_MACHINE\Software\Test]

若要以 .reg 來刪除登錄值 (Value)，請在 DataItemName 的等號後加上減號

"TestValue"=-

Export

有時發現執行了 .reg 但沒有任何改變，可以在 Key 上右鍵 export 出它的 .reg 來查看，可能會是 path 打錯了

御載軟件

御載軟件後，想把所有屬於它的登錄檔都鏟走，可在下面兩個地方做

1) HKEY_LOCAL_MACHINE → Software, and identify the program then right click and delete
2) HKEY_CURRENT_USER → Software, and identify the program then right click and delete
