HardDisk

Definition

● Sector : Minimum track partition for data storage.
● Cluster : Group of sectors.
● Track : One cycle on the platter.
● Cylinder : the same track (top and bottom of every platter) for all platters.

壞軌症狀

● 開檔案或運行軟件速度變慢；
● 聽到硬盤有"嗒嗒"聲；
● Windows 顯示 "無法讀取或寫入"；
● 格式化不成功；
● 每次使用都自動運行 Scandisk 掃描磁片；

壞軌分類

● 邏輯壞軌：磁軌上校驗資訊 (ECC) 和伺服資訊對不上所致。出現這一故障通常是因為一些程式的錯誤操作，也有可能是該處的磁介質開始出現不穩定。
● 物理壞軌：又稱作硬壞軌，是碟片上產生物理變化，磁頭無法正常讀取資料或讀取時間過長。物理壞軌會使磁頭定位錯亂，如果繼續使用，壞軌會逐步影響周邊磁軌，使損壞範圍擴大。

三. 低階格式化

修復工具主要是把出現壞軌位置的空間標記為不可用，進行了一定程度上的隔絕。如果使用 HDD-Scan 掃描的話，這些壞軌還是會顯示出來，想要徹底解決問題，只能是進行低階格式化。


大概原理是將磁片內容清空，恢復出廠狀態，重新劃分柱面和磁軌，再將磁軌劃分為若干個磁區，每個磁區又劃分標識區 ID、間隔區 GAP 和資料區 DATA 等。在劃分的過程中，如果某個存儲位置不可存儲或者不穩定，低階格式化會略掉該位置，讓 Windows 系統或者 HDD-Scan 之類的軟件無法再訪問該位置。


同時，低階格式化是一種損耗性操作，對硬碟的壽命有一定的負面影響。
