Windows 指令

Routing

Add Route

route add des-network mask subnet-mask gateway

Check Route

route print [-4]

[4] Specifies the IPv4 routing table

Delete Route

route delete des-network



Network

tracert

找出由本機到 des-ip 沿途所經過的所有路由器

tracert des-ip

nslookup

DNS query

nslookup www.yahoo.com.tw

Server：DNS server 名稱
Name：yahoo 伺服器的真正名字
Alias：yahoo 伺服器的別名

arp

找出本機中 IP 與 MAC 的配對紀錄：

arp -a

配合 for 功能找出 LAN 內的所有 devices：

for /L %i in (star, step, end) do command

實例：

arp -d
for /L %i in (1,1,254) do ping 192.168.1.%i -n 1 -w 200

arp –d：本機可能有舊紀錄，準確起見先把所有 arp 紀錄刪除
(1，1，254)：由 1 ping 到 254，每次加 1
–n 1：每次發 1 個 echo request
–w 200：200ms to wait for each reply



Remote

Telnet

telnet

Step 1) "Telnet Client" must be enabled in Windows Features.
Step 2) "Telnet Server" also must be enabled if you want someone access this computer.
p.s. Windows 7 can be a "Telnet server", while Windows 10 can't be "Telnet server" in any native way, you have to install third-party apps to start serving Telnet on port 23.

SSH

ssh -l peter 192.168.1.1

Windows 7 to be "SSH Client" the best way is PuTTY. To be "SSH Server" you should download and install "OpenSSH for Windows".

Windows 10 to be "SSH Client" or "Server" is so easy : Settings → Apps & features → Manage optional features → Add feature → OpenSSH Client



Find

很好用的查找功能，to show IPv4 routes with Metric "5256"：

route print -4 | find "5256"

Show all the dynamic ARP records：

arp -a | find "dynamic"



Runas (sudo)

runas /user:user-name program

For example：

runas /user:administrator cmd
