Step2. Port Forwarding

Set the port forwarding rule on your router so the host IP can be reached through the Internet, like this

For more security you could change the port to any other else, from 49152 - 65535
Registry Editor → HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Terminal Server\WinStations\RDP-Tcp → PortNumber → 50001
