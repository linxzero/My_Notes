4. Storage Space & Data Deduplication

Storage Space

所有 SSD，HDD，USB，SAS 大集合，成為一個 storage pool，再用它生成 virtual disk (or storage space)，之後再分盤。


Physical disks ⇒ Storage pool ⇒ Virtual disk ⇒ Disk drive


Enterprise storage needs

Storage space 事關重大，所以你可以采用以下方案來保障服務質素：
● Mirror/Parity support (RAID 1)
● Data Stripping (RAID 0)
● Enclosure Awareness
● Storage Tiering (分 SSD 和 HDD 兩層，常用放在 SSD，不常用放 HDD，詳細看 L4-24)
● Storage Replication
● Data Deduplication
● Data Encryption
● Performance Analysis
