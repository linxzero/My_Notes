2. Local Storage

Partition Table Format

MBR (master boot record)

自1980s，最多 4 個 primary partition，每個 partition 最大 2TB。

GPT (globally unique identifier)

自2000s，最多 128 個 partition，每個 partition 最大 18EB。 (GB→TB→PB→EB)

Disk Type

Basic disk

正常用法，可以分 partition。

Dynamic disk

可以在電腦正在運行的時侯設置 RAID，在 disk manager 入面 set。實際使用只比在 BIOS 設置的 RAID 慢少許。Dynamic disk 分的叫做 volume 而不是 partition。
Basic disk 可以轉成 Dynamic disk，但 Dynamic 就不能轉成 Basic。
