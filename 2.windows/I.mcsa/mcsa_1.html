1. Upgrade & Migrate

Server 2016 edition

1) Essentials SBS (small business server)
2) Standard 1P + 2V  (P is physical host, V is VM)
3) Datacenter 1P + unlimited-V
4) Hyper-V Microsoft Hyper-V Server 2016, free download for testing hyper-v, no GUI.
5) Storage Workgroup Used as fileserver (NAS) with windows GUI.
6) Storage Standard More function than storage workgroup, such as DNS server.

System Installation Option

1) Full Server Installation With desktop experience.
2) Server Core Installation No GUI, less ram required, less attacking risk, but some apps with GUI can’t be installed.
3) Hyper-V Container Nano server. No GUI, no CLI, seldom used, transformed to container already.

Remote Management

1) RSAT (Remote Server Administration Tools)
2) Windows Powershell
3) Remote desktop
4) GPO (group policy), centralized control in same domain.

Service Channel

(kind of update channel, available for all editions)

1) SA channel (semi-annual) update feature every 6 months.
2) LTSC (long term service channel)

Solution Tools

在你為公司的系統升級前，你必須先了解升級的可能性，以及升級後對舊有軟件的影響，這時你需要一些工具來幫你分析。

1) MAP (Microsoft Assessment and Planning toolkit)
Toolkit for windows server 2016, check all PCs’ enviroment, check what problem would be issued when upgrading server 2016.


2) MDT (Microsoft Deployment Toolkit)
That's free from microsoft, task sequencing, different apps installed in different image.
