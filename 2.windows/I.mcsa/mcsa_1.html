1. Upgrade & Migrate

Server 2016 edition

● Essentials SBS (small business server)
● Standard 1P + 2V  (P is physical host, V is VM)
● Datacenter 1P + unlimited-V
● Hyper-V Microsoft Hyper-V Server 2016, free download for testing hyper-v, no GUI.
● Storage Workgroup Used as fileserver (NAS) with windows GUI.
● Storage Standard More function than storage workgroup, such as DNS server.

System Installation Option

● Full Server Installation With desktop experience.
● Server Core Installation No GUI, less ram required, less attacking risk, but some apps with GUI can’t be installed.
● Hyper-V Container Nano server. No GUI, no CLI, seldom used, transformed to container already.

Remote Management

● RSAT (remote server administration tools)
● Windows Powershell
● Remote desktop
● GPO (group policy), centralized control in same domain.

Service Channel

(kind of update channel, available for all editions)

● SA channel (semi-annual) update feature every 6 months.
● LTSC (long term service channel)

Solution Tools

在你為公司的系統升級前，你必須先了解升級的可能性，以及升級後對舊有軟件的影響，這時你需要一些工具來幫你分析。

● MAP (Microsoft Assessment and Planning toolkit)
Toolkit for windows server 2016, check all PCs’ enviroment, check what problem would be issued when upgrading server 2016.


● MDT (Microsoft Deployment Toolkit)
That's free from microsoft, task sequencing, different apps installed in different image.

Migrating Server

● ADMT (active directory migration tool)
When migrating server roles across domain of forest, use tool ADMT, that's free.

Activation

● KMS 舊款機用，先起一個 KMS host，然後 KMS client 會自己去搵 host 激活，免卻了每部機都去搵 Microsoft 來激活。有最低消費，最少要激活 5 部 server 和 25 部 client。
● AD-Based 新款機可以用，用 AD DS (active directory domain services) 來儲存 activation objects，然後 clients 自己搵 AD DS 獲得 activation objects，同樣免卻了每部機搵微軟。激活的機要是 Windows Server 2012 以上，或 Windows 8 以上。而 AD DS schema 要是 Windows Server 2012 以上。
● MAK (multiple activation key)，其實是一條 volume license key，當你公司的電腦有很多不在公司，分散在各地，或者當地沒有網絡，就可用這方法。
