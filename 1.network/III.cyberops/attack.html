Network Attack

Reconnaissance Attack

勘察，目的是找尋容易攻擊的對象。

● Internet information query (上網查關於你的資料，例如用 whois.net)
● Ping sweep
● Port scan

Access Attack

目的是偷資料，或者奪掌控權。

● Password attack (比如用 phishing, dictionary attacks, brute-force attacks, network sniffing)
● Pass-the-hash (偷 hash)
● Trust exploitation
● Port redirection
● Man-in-the-middle attack
● IP, MAC, DHCP Spoofing

DoS Attack

Evasion Methods

(隱身術)

Encryption and tunneling

加密 malware，或者加擾碼落去 malware (這動作叫 scramble)，使其難以被偵測。

Resource exhaustion

用到你盡，等你忙到吾得閒查有冇入侵者。

Traffic fragmentation

逐少偷運，將 malware 斬件，分多次偷運，等你吾察覺。

Protocol-level misinterpretation

把 malware 放在 checksum 或 TTL value 等地方中偷運，瞞騙 firewall。

Traffic substitution

偽妝入境，用不同格式對文件進行編碼，瞞騙 IPS。

Pivot

以一部失守的主機，嘗試進入周圍更多的設備。

Rootkit

通過修改因運行程式所帶來的一切資料，比如行程、登錄檔和系統紀錄檔等，來抹去一切痕跡，隱藏應用程式。
