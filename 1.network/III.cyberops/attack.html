Attack

Network Attack

Reconnaissance Attack

勘察，目的是找尋容易攻擊的對象。

● Internet information query (上網查關於你的資料，例如用 whois.net)
● Ping sweep
● Port scan

Access Attack

目的是偷資料，或者奪掌控權。

● Password attack (比如用 phishing, dictionary attacks, brute-force attacks, network sniffing)
● Pass-the-hash (偷 hash)
● Trust exploitation
● Port redirection
● Man-in-the-middle attack
● IP, MAC, DHCP Spoofing
