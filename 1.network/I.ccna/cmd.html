Initial setting

no ip domain-lookup
hostname R1
banner motd % Welcomm!! %

Console password

line console 0
password 88888
login

Virtual terminal password

line vty 0 15
password 88888
login
transport input telnet

Privileged mode password

enable password 88888
enable secret 99999

PS : According to the setting above, "99999" will replace "88888" to be the only password to access privileged mode.
