Cmd 匯總

Initial setting

no ip domain-lookup
hostname R1
banner motd % Welcomm!! %

Console password

line console 0
password 88888
login

Virtual terminal password

line vty 0 15
password 88888
login
transport input telnet

PS :

  • Mostly, the default transport input protocol is "telnet".
  • Before you can remotly access, "Enable password" has to be set.

Enable password

enable password 88888
enable secret 99999

PS : According to the setting above, "99999" will replace "88888" to be the only password to reach privileged mode.

NTP

ntp server time.google.com
ntp peer 10.1.1.20

PS : 只有與 server 失去聯繫時，才會向 peer 取暖。

show ntp status
show ntp associations
show clock

PS :

  • "show ntp status" 顯示 server 位置、stratum.
  • "show ntp associations" 顯示 peer.
  • Stratum 是層數 (0 – 16)，0 代表時間源頭，16 代表無法同步。

clock set hh:mm:ss 08 jun 1988
ntp master 5

PS : "5" 的意思為向下校時最多到第 5 層。

Syslog

logging 10.1.1.2
logging on
logging trap severity-level

PS : "severity-level" is 0 – 7
