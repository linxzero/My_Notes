Switch

2960 Switch LEDs

Name Stand for Indicator Explain Port LED Explain
SYST System off system is not powered on – –
green system is operating normally – –
amber system is not functioning properly – –
RPS Redundant Power System off RPS is off – –
green RPS is connected but is unavailable because it is providing power to another device – –
amber RPS is in standby mode – –
blinking amber the internal power supply in the switch has failed, and RPS is providing power – –
STAT Port Status green the port status mode is selected off no link or port is adminly shut down
green link is present
blinking green there is activity and the port is sending or receiving data
green-amber link fault
amber port is blocked to ensure that a loop does not exist in the forwarding domain and is not forwarding data
blinking amber port is blocked to prevent a possible loop in the forwarding domain
DUPLX Port Duplex green the port duplex mode is selected off in half-duplex mode
green in full-duplex mode
SPEED Port Speed green the port speed mode is selected off 10Mb/s
green 100Mb/s
blinking green 1000Mb/s
PoE Power over Ethernet Mode off the PoE mode is not selected and that none of the ports have been denied power or placed in a fault condition – –
blinking amber the PoE mode is not selected but at least one of the ports has been denied power, or has a PoE fault – –
green the PoE mode is selected off PoE is off
green PoE is on
green-amber PoE is denied because providing power to the powered device will exceed the switch power capacity
blinking amber PoE is off due to a fault
amber PoE for the port has been disabled
