● 暫停命令可以按：

Ctrl-Shift-6

  • "Unreachable", when a host or gateway receives a packet that it cannot deliver, it uses an ICMP Destination Unreachable message to notify the source that the destination or service is unreachable.
  • "Time Exceeded", if a router receives a packet with TTL is 0, it discards the packet and sends a Time Exceeded message to the source host.
