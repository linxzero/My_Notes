Routing (IPv6)

Static Route IPv6

ipv6 unicast-routing
ipv6 route 2002:DB8:2de:0:0:0:0:e13/64 g0/0

RIPng

ipv6 unicast-routing
do show ipv6 int br
int g0/0
ipv6 rip cisco enable
ipv6 rip cisco default-info

"cisco" is RIP process name.

EIGRP IPv6

ipv6 unicast-routing
ipv6 router eigrp 2
no shutdown
eigrp router-id 1.1.1.1

int e0/0
ipv6 eigrp 2

"router-id" configuration is required.

OSPFv3

ipv6 unicast-routing
ipv6 router ospf 10
router-id 1.1.1.1
do clear ipv6 ospf process

int e1/1
ipv6 ospf 10 area 0
