Vlan

Access port

int range f0/1-10
switchport mode access
switchport access vlan 10

By default, all ports access vlan 1.

Trunk port

int g0/1
switchport mode trunk
switchport trunk native vlan 99

Verifying

show vlan brief
show int trunk
show int switchport
show int f0/1 sw

● "vlan br" show all vlans.
● "f0/1 sw" show the switchport mode.
