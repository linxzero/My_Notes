SPAN

(Switched Port Analyzer)
It's a port mirroring feature on Cisco switches.

Term Definition
Ingress traffic 進入 switch 的 traffice
Egress traffic 離開 switch 的 traffice
Source port 受監視的 port。這個 port 的所有進出 traffic 都會 copy 一份去 destination port
Destination port 直連着 analyzer 的 port
SPAN session An association between source and destination port
Source VLAN 受監視的 VLAN

注意 3 點：
● Destination port 不能為 source port.
● Destination port 數量 由 switch platform 決定。
● Destination port 做不了正常 switch port.

monitor session 1 source int f0/1
monitor session 1 destination int f0/2

● f0/1 的所有進出 traffic 都會 copy 一份去 f0/2.

show monitor

RSPAN

(Remote SPAN)
當 source port 和 destination port 不在同一個 switch 時，用 RSPAN。

● 在所有途經的 switches 都需要定義一個 VLAN 來傳送 RSPAN traffic.
● RSPAN 在 CCNA Security 內詳談。
