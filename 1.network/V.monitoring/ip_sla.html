IP SLA

(IP Service Level Agreement)

IP SLA 會不間斷 generate traffic 來實時監測網絡的表現。例如：不斷 ping DNS server 來監察與 server 的連線情況。

ip sla 1
icmp-echo 192.168.1.1
frequency 30

● "1" is the operation number.
● "icmp" 不停 ping 192.168.1.1 (gateway).
● "frequency" 每 30s 做一次。

ip sla schedule 1 start now life forever

● "now" sla operation number 1 即時生效。
● "forever" 直到手動停止。


show ip sla configuration
no ip sla schedule
