IGP and EGP

IGP

(Interior Gateway Protocol)
RIP, EIGRP and OSPF 就屬於 IGP，在一個 AS (Autonomous System) 內交換 routing information. 比如用在一間企業的內部網絡。

EGP

(Exterior Gateway Protocol)
BGP 屬於 EGP，用於在不同 AS 之間交換 routing information. 比如 ISP 之間的路由交換。

IGP and EGP routing protocols

BGP

iBGP

(Internal BGP)
在相同 AS 內的 routers，用 iBGP。

eBGP

(External BGP)
在不同 AS 之間的 routers，用 iBGP。CCNA 只講 eBGP。


router bgp 65500
neighbor 209.16.21.1 remote-as 65501
network 198.133.9.0 mask 255.255.255.0

● "65500" 是自己 AS number.
● "neighbor" 指示鄰居的 IP 和 AS number.
● "network" 把這段 network 告訢鄰居。

Verify

show ip bgp summary
